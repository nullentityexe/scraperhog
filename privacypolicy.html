ScraperHog

ScraperHog does not collect any data from it's users. The only data collected is from the sites in which the user chooses to scrape.

The data and requests made are not stored anywhere within ScraperHog's files.

ScraperHog functions through the platform Discord. A discord account is needed to use ScraperHog, but no information about the account issuing commands is stored.

Information gathered by ScraperHog may be displayed on Discord, but it is not stored or recorded anywhere within ScraperHog

ScraperHog may process any information you give it in Discord (i.e. url), but it is real time and never stored.

ScraperHog uses the Administrator permission within discord so that it is able to send and receive commands simply, without the hassle of missing permissions. ScraperHog cannot and will not view or record any information about the users using it or the server it is used in.


This privacy policy may be updated from time to time
