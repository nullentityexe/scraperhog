ScraperHog

ScraperHog is a lightweight discord bot for scraping the web in the comfort of discord.

To keep ScraperHog as lightweight as possible, it only has 2 commands

/info

No parameters

Provides information on the current version of ScraperHog and a brief description

/scrape

1 parameter: url

Connects to the website through the url provided to initiate scraping.

The response returns the title of the website as well as two options

-Fetch Links

Fetch Links returns a reponse containing a text file of all scraped links.

-Snapshot

Snapshot takes two screenshots of a website: a typical page screenshot, in typical dimensions, and a full page screenshot


These commands & options allow you to fully explore the web through ScraperHog


Run ScraperHog on an url, and then take a screenshot, or return urls. You can continue crawling by running the /scrape command and inputting one of these urls
